# Bundled assets

These ship inside the add-in (copied to `dist/assets/` by `tools/build-web.mjs`)
and are served from the project's own hosting — **never** fetched from a third
party at runtime (so no document content or usage leaks; see
`docs/phase2/SECURITY.md`).

## Font — `NotoSansBengali.woff2`

- **What:** Noto Sans Bengali (variable, Bengali subset, weights 400–700). The
  single canonical Unicode output font (decision D-0005).
- **Source (pinned):** `@fontsource-variable/noto-sans-bengali@5.2.5`
  via `https://cdn.jsdelivr.net/npm/@fontsource-variable/noto-sans-bengali@5.2.5/files/noto-sans-bengali-bengali-wght-normal.woff2`
- **SHA-256:** `0f9b92da1cb4a7a15a79172854f37aa0f5606b10b47568a0bc2f561af481a6a8`
- **Licence:** SIL Open Font License 1.1 — see `NotoSansBengali-OFL.txt`
  (official, from `google/fonts`). Noto Sans Bengali has **no Reserved Font
  Name**, and this file is **unmodified** (not subset/renamed by us), so
  redistribution simply requires shipping the OFL text alongside it (done).

To re-verify: download the pinned URL and confirm the SHA-256 matches.
